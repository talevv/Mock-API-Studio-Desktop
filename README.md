# Mock API Studio

A desktop application for creating and managing mock API servers.

## Overview

Mock API Studio is an Electron-based desktop application designed to simplify the process of setting up mock servers for API testing and development. It provides an intuitive interface for defining endpoints, configuring responses, and managing mock server instances without the need for complex server setup.

## Purpose

This project was created primarily for **learning purposes**, serving as an exploration of:

- **Electron** - Building cross-platform desktop applications
- **Redux Toolkit** - Modern Redux state management with simplified API
- **RTK Query** - Advanced data fetching and caching layer
- **SQLite** - Lightweight database integration for persistent storage
